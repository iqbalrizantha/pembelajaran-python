Pembelajaran Python

Apa itu Python?
Python adalah bahasa pemrograman tingkat tinggi yang mudah dipelajari dan digunakan secara luas, cocok untuk pemula maupun profesional.

Sejarah Singkat:
Diciptakan oleh Guido van Rossum dan pertama kali dirilis pada tahun 1991.

Implementasi:
Python bisa digunakan untuk web development, data science, AI, otomasi, dan lainnya.

Contoh Program Sederhana:

print("Halo Dunia!")
for i in range(3):
    print("Belajar Python itu menyenangkan!")
× Mulailah belajar Python dengan memahami sintaks dasar seperti print(), variabel, dan perulangan.
